# Bootcamp-Fundamentos-de-Engenharia-de-Dados-e-Machine-Learning
Bootcamp focado em construir uma base sólida em Python e bancos de dados, entender os fundamentos de ETL e cloud, e evoluir até Machine Learning — aplicando cada conceito em projetos que simulam desafios reais do mercado de dados, com uma trilha que já inclui IA generativa e integração de LLMs em pipelines..
